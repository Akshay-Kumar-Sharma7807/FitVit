# **App Name**: FitVit

## Core Features:

- Diet Guide: AI-Powered Personalized Diet Guide: Users set wellness goals (e.g., muscle building, skin health) and receive customized meal suggestions and recipes.
- Smart Shopping: Automated Supplement Ordering Tool: Based on the user's diet plan, the tool automatically suggests and orders supplements and groceries from online retailers.
- User Profile: Profile and Goal Setting: UI to create user profiles, set wellness goals, and track progress.
- AI Workout Plan Generator: AI workout Plan generator: An AI Tool will suggest and display to the user tailored short workout video routines including Yoga, or stretches
- Community UI: Community Recommendation System: display and allows users to access virtual fitness classes, therapy sessions, or wellness communities that match their goals.
- Calendar progress View: Weekly and Monthly progress views with a progress calendar showing attended fitness classes and diet progress with options for each for adjusting plans or editing entries.  

## Style Guidelines:

- Primary color: Soft teal (#79D0C3) to evoke calmness and well-being.
- Background color: Light gray (#F0F4F3), almost white but not quite, for a clean and calming background.
- Accent color: Warm coral (#F0795A) to highlight important elements and CTAs.
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of text, and 'PT Sans' (sans-serif) for body text, providing a modern and readable interface.
- Use minimalist, rounded icons representing diet, exercise, and mental wellness.
- Clean and spacious layout with a focus on readability and easy navigation.
- Subtle transitions and animations to provide feedback and enhance user engagement.
